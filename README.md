## Usage
1. This Tool is CommandLine App.
2. Arguments are x,y ranges, Sensor_ip, Sensor_port

```
~/App/current/path.exe [minXpos] [maxXpos] [minYpos] [maxYpos] [SensorIP] [SensorPort]
```
3. Pos's unit is millimeter
